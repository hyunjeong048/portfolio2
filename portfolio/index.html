어바웃

AI가 빠르게 발전하며 웹 환경이 급변하는 지금, REACT 단순한 코딩을 넘어 ‘생각하는 퍼블리셔’가 필요하다고 느꼈습니다.
저는 변화에수동적으로 적응하기보다, AI 와 새로운 기술을 적극적으로 활용하여 더 나은 웹 환경을 만들어가는 성장하는 퍼블리셔를 지향합니다.
자동화된 코드 CAFE24 속에서도 사용자의 경험을 세밀히 읽고, 디자인의 의도를 접근성 높은 JAVASCRIPT 마크업과 동적UI로 정확히 구현하는 것이 제 역할이라 생각합니다. 디자인과 개발을 잇는 Interactive 징검다리로서 원활한 협업을 이끌고, 다양한 아이디어와 의견을 조율하여 건강한 시너지를 창출하는 웹 퍼블리셔가 되겠습니다. 새로운 기술을 두려워하지 않고 창의적으로 DEVELOP 문제를 해결하며, 능동적인 자세로 VUE.JS 최적화된 웹 경험을 제공하기 위해 끊임없이 성장하겠습니다.
